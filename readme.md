# Node API boilerplate

This is our boilerplate project for an awaosme nodejs api.

## Getting Started

Install dependencies.

```
npm i
```

Then using Nodemon

```
nodemon
```

### Prerequisites

Please install Node with npm.

- [Nodejs](https://nodejs.org)
